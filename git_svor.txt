Git spurningar og svör

1. Hvað gera eftirfarandi Linux skipanir?
	cd = breit directories
	ls = sínnir allar directroies
	pwd = sinnir directroies sem þu ert að nota og full skrar leið
	mkdir = býr til directorie

2. Hvað gera eftirfarandi git skipanir og hvers vegna eru þær gagnlegar?
	git clone = klonar repository í nýtt directory
	git log = sýnnir commit logs
	git status = sýnnir okkur leiðir sem hafa mismunir á index og head commit
	git diff = sýnnir ok hluta sem hafa breist index og full anað
	git checkout = uppfærir hluti

3. Hver er munurinn á eftirfarandi git skipunum:
	a) git diff 
	b) git diff --staged 
	c) git diff commit1 commit2
	svar = margt og mikið

4. 	Hvað er útgáfustýring (e.version control)? 
	Það geymir allar útgáfur af verkefni og leyfir manni að hoppa á milli þeirra

5.	Hverjir eru helstu kostir við að nota GIT?
	Að vinna með fólki á milli tölva

6.	Hversu oft telur þú að eigi að gera commit í verkefni, rökstuddu?
	Eins oft og maður getur og það er ekki að hægja á "workflow" mans

7.	Hvað er "Working directory"?" "Staging area"?" og "Repository" í GIT?
	Working Directory: Núverandi vinnusvæði
	Staging Area: Skrá sem geymir upplýsingar um verkefnið
	Repository: Verkefnis mappan

8.	Hvenær er sniðugt að nota greinar (branches)? 
	Þegar þú ert ekki viss um breytingar se mþú ætlar að gera

9.	Hver fann upp GIT?
	Linus Benedict Torvalds

10.	Hvar er GIT helst notað?
	Forritun og fleirri tölvutengd verkefni
